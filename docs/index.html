Alerts

Alerts are short messages used to provide important information to the users for his actions in the website.

Basic alerts

Baisc alerts contains only text. In simple-ui we provide you with 6 different alerts and can be used by adding classes (.alert-success, .alert-warning etc).

Avatar

Avatars are used across website to display profile pictures in different places like profile, chat and posts.

Avatar with image

Avatar with image can be used by adding avatar-img class to image tag and we provide you with 4 different sizes for different pages. 

          
            <img src="../../images/profile.jpeg" class="avatar-img avatar-lg" />
            <img src="../../images/profile.jpeg" class="avatar-img avatar-md" />
            <img src="../../images/profile.jpeg" class="avatar-img avatar-sm" />
            <img src="../../images/profile.jpeg" class="avatar-img avatar-xs" />

Avatar with text

Avatar with text can be used in absense of image to display user initial etc. Add class avatar-text to div containing text to make avatar with text.

Badge

Badges are used to display info relative to parent components. Information like notification count and status are provided.

Badge on avatar

Badge on avatar can be added by wrapping avatar inside div with class name badge.
