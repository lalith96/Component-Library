Alerts

Alerts are short messages used to provide important information to the users for their actions on the website.

Basic alerts

Basic alerts contain only text. In Simple-UI we provide you with 6 different alerts and can be used by adding classes .alert, .alert-success, .alert-warning etc.

Alert with text message and primary color.
Alert with text message and secondary color.
Alert with success text message.
Alert with warning text message.
Alert with information.
Alert with dark background.

Avatar

Avatars are used across websites to display profile pictures in different places like profiles, chats and posts.

Avatar with image

Avatar with an image can be used by adding .avatar-img to the image tag and Simple-UI provides 4 different sizes for different pages.

            <img src="/images/profile.jpeg" class="avatar-img avatar-md" />
            <img src="/images/profile.jpeg" class="avatar-img avatar-sm" />
            <img src="/images/profile.jpeg" class="avatar-img avatar-xs" />

Avatar with text

Avatar with text can be used in the absence of the image to display user initials etc. Add .avatar-text to div containing text to make an avatar with text.

Badge

Badges are used to display info related to parent components. They provide information like notification count and status.

Badge on avatar

A badge on the avatar is displayed by wrapping the avatar inside div with .badge class added to it. Add class with width and height same as avatar.

Buttons

Buttons are used to perform actions in websites, and they are seen across the website on almost all pages.

Buttons with different variations

Simple UI provides 6 different variations of buttons. Some buttons used for specific purpose while other can be used anywhere. Add .btn and btn-primary, btn-success etc for different styles in your code. 

          
            <button class="btn btn-primary">Primary</button>
            <button class="btn btn-secondary">Secondary</button>
            <button class="btn btn-success">Success</button>
            <button class="btn btn-warning">Warning</button>
            <button class="btn btn-info">Info</button>
            <button class="btn btn-dark">Dark</button>

Buttons with icons

These icons make buttons easily understood by anyone. Just add icon inside button tag. 

          
            <button class="btn btn-primary">
                <i class="fa-solid fa-circle-xmark"></i>
                Wrong
            </button>
            <button class="btn btn-success">
                <i class="fa-solid fa-circle-check"></i>
                Success
            </button>
            <button class="btn btn-warning">
                <i class="fa-solid fa-triangle-exclamation"></i>
                Warning
            </button>
            <button class="btn btn-info">
                <i class="fa-solid fa-circle-info"></i>
                Info
            </button>

Buttons with links

These button with links can be used as mostly secondary action buttons or in place of links. Add .btn-link class to the button tag in your code. 

          
            <button class="btn-link btn-link-primary">Link</button>
            <button class="btn-link btn-link-success">Success</button>
            <button class="btn-link btn-link-info">Info</button>

Floating action button

Floating action button is used for important actions in website like move to top, opening camera etc. For button tag add .btn-floating class and add icon of your choice inside it based on requirement. 

          
            <button class="btn btn-floating btn-primary">
              <i class="fa-solid fa-plus"></i>
            </button>

Image

Images make a website more attractive and adds value when presenting important information. Simple UI provides responsive images and round images.

Responsive Image

Responsive image fits to parent's width and height and it adjusts according to screen size. Add .image-responsive to the img tag in your code. 

          
            <img src="../../images/image.jpeg" class="image-responsive" />

Round Image

Round Image also derives width and height from parent class and adjusts according to screen size. Add .round-image to the img tag in addition to .image-responsive to make it responsive and round.
Note: It should always be wrapped in a div with specifc width. 

          
            <img src="../../images/profile.jpeg" class="image-responsive round-image" />
